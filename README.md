# randomxd

Pick a random movie from your Letterboxd watchlist.

## Setup

1. Get a free API key from [TMDB](https://www.themoviedb.org/settings/api)
2. Copy `.env.example` to `.env.local` and add your key
3. Install dependencies and run:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000), enter a Letterboxd username, and hit "Pick a movie".
